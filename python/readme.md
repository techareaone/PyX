This folder contains the Python source file for PyX as it is intended to be run directly. If you have Python installed along with the required dependencies (`customtkinter`, and optionally `cryptography`), you can launch PyX by running the script inside this folder rather than using the compiled executable. This is useful for development, debugging, or inspecting the code before running it. The source here reflects the exact logic shipped in the release executable — nothing is stripped or modified. For normal use, the executable is recommended. For transparency or contribution purposes, this is the canonical runnable source.
